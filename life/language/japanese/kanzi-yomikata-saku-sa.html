    作 (さく, さ) 한자 읽는법 読み方 | 일본어 공부

    2022-04-17 life / language / japanese

    ToC on/off

    개요

    일본어 한자 作는 作る처럼 훈독의 경우에는 つくる가 되지만, 음독의 경우는 さく, さ 두가지 방법 중에서 대부분 さく로 읽게 되고, 몇가지 특수한 경우에만 さ로 읽게 되는데 대표적인 단어를 각각 정리해보겠습니다.

    作 (さ)로 읽는 경우

    さ로 읽는 경우는 다음과 같은 단어에 사용될 때입니다.

    • 作業(さぎょう) : 작업
    • 作動(さどう) : 작동
    • 作用(さよう) : 작용
    • 操作(そうさ) : 조작
    • 動作(どうさ) : 동작
    • 発作(ほっさ) : 발작
    • 無造作(むぞうさ) : 손쉽게, 아무렇지 않게 하는 모양
    • 作法(さほう) : 규범

    作 (さく)로 읽는 경우

    さく로 읽는 경우는 다음과 같은 단어에 사용될 때입니다.
    거의 대부분이 さく로 읽기 때문에 위에서 나온 さ로 읽는 경우 몇가지를 제외하면 특별히 신경쓰지 않아도 될 정도입니다.

    • 作詞(さくし) : 작사
    • 作者(さくしゃ) : 작자
    • 作成(さくせい) : 작성
    • 作品(さくひん) : 작품
    • 工作(こうさく) : 공작
    • 製作(せいさく) : 제작
    • 創作(そうさく) : 창작
    • 名作(めいさく) : 명작
    • 作戦(さくせん) : 작전
    • 耕作(こうさく) : 경작
