    네이버 클라우드 (Ncloud) 자격증 NCA 합격 후기, 시험문제 출제 경향

    네이버 클라우드 플랫폼 (Ncloud) 자격증 NCA 시험 합격 후기와 시험문제 출제 경향을 정리해보았습니다.

    NCA 합격

    다소 걱정했던 네이버 클라우드 (Ncloud) 자격증 NCA (NAVER CLOUD PLATFORM Certified Associate) 시험에 드디어 합격했습니다.

    설마 이런 것까지 문제로 나오는가 싶은 것들도 많아서 생각했던 것 보다는 다소 어렵게 느껴습니다.

    혹시 7월 말에 네이버 클라우드 최고 등급 자격증인 NCE (NAVER CLOUD PLATFORM Certified Expert) 시험이 시작되었다고 하던데 그 영향인지, 아니면 제가 8월 초에 시험을 봤는데, 월초 시험이라 좀 더 어렵게 나온 것은 아닐까 하는 생각도 들었지만 근거는 없습니다.

    출제 경향

    제가 그닥 암기를 잘하는 것도 아니어서 시험 문제를 잘 기억하지 못하기도 하고, 기억 한다고 해도 함부로 올려도 되는지 확실치가 않아서 대략적인 출제 경향만 적어 보겠습니다.

    • Classic 환경보다는 VPC 환경에서 훨씬 많이 나옵니다. VPC 환경 위주로 공부하시길 추천 드립니다.
    • VPC, Subnet, ACG의 기본 설정 관련 내용과 수치를 묻는 문제도 나옵니다.
    • Auto Scaling 관련 문제도 많이 나옵니다.
    • Auto Scaling의 Launching Configuration과 Auto Scaling Group 설정에 각각 어떤 것들이 있는지 비교하는 문제가 나옵니다.
    • Load Balancer 설정과 특징 관련 문제도 꼭 나옵니다.
    • 각 상품이 어떤 서비스 카테고리에 속하는지 묻는 문제도 나옵니다.
    • 상품의 이용 요금 계산 방식을 묻는 문제도 나옵니다.
    • Object Storage, CDN 관련 문제도 많이 나옵니다.

    시험 합격을 위한 Tip

    1. 네이버 클라우드 (Ncloud)에서 제공하는 자격증 공인 교육을 꼭 들으시기 바랍니다.
      공인 교육은 무료와 유료가 있는데, 가능하면 유료 강의를 듣는 것을 추천합니다.
      그리고 무료와 유료 모두 강의 끝에 시험 예상 문제를 몇 문제 풀이하는 시간이 있는데 꼭 외우시기 바랍니다.

    2. 네이버 클라우드 (Ncloud) 홈페이지 서비스 소개와 가이드 문서를 꼼꼼히 읽어 보시길 추천합니다.
      가이드 문서 중에서도 [주의], [제한 사항] 등의 항목들은 출제 확률이 높으니 자세히 보시길 추천합니다.

    3. 내용을 그냥 외우는데 그치지 말고, 가능하면 직접 예상 문제를 만들어서 모의 시험을 보는 것처럼 연습해보시길 추천합니다.

    4. 시험을 볼 때 모르는 문제는 건너띄거나, 아무거나 찍은 다음에 문제를 다 풀고 나서 다시 돌아와서 천천히 살펴보는 것을 추천합니다. 60문제에 60분 제한 시간이므로 1분에 1문제인데 생각보다 시간은 여유가 있습니다. 확실하고 쉬운 문제부터 체크하고 어렵거나 헷갈리는 문제는 나중에 푸는 것을 추천합니다.
