    Jekyll Helium 테마 소개

    Jekyll의 인기 테마 중에서 비지니스, 홈페이지용으로 사용되는 Helium 테마를 간단히 소개해보겠습니다.

    ToC on/off

    개요

    Jekyll에서 사용하는 여러 종류의 테마 중에서 비지니스용 (회사 홈페이지, 랜딩 페이지) 테마에서 인기 순위 상위권을 차지하는 Helium 테마를 소개해보려고 합니다.

    구성 요소

    Helium 테마의 특징적인 구성 요소들을 정리해보면 다음과 같습니다.

    저는 이 중에서 100개 이상의 다양한 UI 블럭이 Helium 테마를 선택해야 하는 가장 큰 장점이 아닐까 생각합니다.

    메인 요소

    • Bootstrap 4
    • Font Awesome
    • 100개 이상의 다양한 UI 블럭
    • 반응형 디자인

    Jekyll 특화 요소

    • Jekyll 3.x 이상, Github 지원
    • SEO 최적화
    • Google Analytics 지원
    • Google AdSense 지원
    • Disqus comments 지원

    기타 요소

    • 블로그 페이지 샘플
    • 랜딩 페이지 샘플
    • 테그 페이지
    • 슬라이딩 배너 지원
    • Emoji 지원
