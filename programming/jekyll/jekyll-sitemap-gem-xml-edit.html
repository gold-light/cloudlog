    Jekyll sitemap.xml 템플릿 수정하기 | Ruby 3.0

    Ruby 3.0 환경에서 sitemap.xml 템플릿 파일 위치에 대한 정보입니다.

    ToC on/off

    사이트 환경

    • Ruby 3.0
    • Jekyll 4.2.2
    • jekyll-sitemap-1.4.0

    파일 위치

    gem 파일 위치는 Ruby 설치 폴더 아래쪽에 있는데, 아래 폴더에 찾아가면 sitemap.xml 파일이 보입니다.

    C:\Ruby30-x64\lib\ruby\gems\3.0.0\gems\jekyll-sitemap-1.4.0\lib

    파일 수정

    기본값은 loc, lastmod 두가지 항목만 설정되어 있는데, image 관련 정보를 추가 한다거나 할 경우에는 아래 내용을 수정하면 됩니다.
