    디아블로 이모탈 클라이언트 설치 용량, 시스템 요구사항 정리

    2022-06-03 game / mobile / diablo-immortal

    개요

    디아블로 이모탈 (Diablo Immortal) PC 버전과 모바일 iOS, Android 버전의 클라이언트 설치 사이즈와 시스템 요구사항에 대해 정리해보겠습니다

    클라이언트 설치 사이즈

    PC 버전

    PC버전의 클라이언트 사이즈는 설치 화면에서 26.80GB인 것으로 확인됩니다.

    iOS 버전

    아이폰 등의 클라이언트 사이즈는 초기 설치 사이즈 3.57GB + 이후 리소스 업데이트 사이즈 8.83GB = 12.40GB 정도 이며 앞으로 계속 업데이트가 되면 늘어날 것으로 예상됩니다.
    확인하는 방법은 아이폰 [설정] - [일반] - [iPhone 저장 공간] - [Immortal]에 들어가시면 확인 가능합니다.

    시스템 요구사항

    PC 버전

    최소 사양

    • OS: Windows® 7 / Windows® 8 / Windows® 10 / Windows® 11 (64-bit)
    • CPU: Intel® Core™ i3 또는 AMD™ FX-8100
    • 비디오카드: NVIDIA® GeForce® GTX 460, ATI Radeon™ HD 6850 또는 Intel® HD Graphics 530
    • RAM: 4GB RAM
    • 인터넷: 광대역 인터넷 연결
    • 해상도: 1920 x 1080 최소 디스플레이 해상도

    권장 사양

    • OS: Windows® 10 / Windows® 11 (64-bit)
    • CPU: Intel® Core™ i5 또는 AMD Ryzen™ 5
    • 비디오카드: NVIDIA® GeForce® GTX 770 또는 AMD Radeon™ RX 470
    • RAM: 8GB RAM
    • 인터넷: 광대역 인터넷 연결
    • 해상도: 1920 x 1080 최소 디스플레이 해상도
