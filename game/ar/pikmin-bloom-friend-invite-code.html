    피크민 블룸 (Pikmin Bloom) 친구 초대 코드 - WGFNROFND

    2022-05-08 game / mobile / ar

    ToC on/off

    개요

    피크민 블룸 (Pikmin Bloom)은 포켓몬고(Pokemon Go)와 인그레스(Ingress)를 제작한 나이앤틱(Niantic)에서 닌텐도의 게임큐브, Wii용 게임 피크민(Pikmin)을 이용해 제작한 모바일 위치기반 증강현실(AR) 게임입니다.

    친구 초대 코드 - WGFNROFND

    피크민 블룸 계정을 처음 만들때 친구 초대코드를 입력할 수 있습니다.
    친구 초대코드를 입력하면 [꽃잎 20장과 정수 10개]를 받을 수 있습니다.

    제가 드리는 초대코드는 [ WGFNROFND ] 입니다.

    보상은 튜토리얼을 완료한 후에 지급되고 자동적으로 친구로 등록되는데, 계속 친구 리스트 등록되어 있는 것이 부담스러우시면 보상을 받은 후에 친구 삭제를 하면 되겠죠.

    다운로드 경로

    • Apple App Store: https://apps.apple.com/app/id1556357398
    • Google PlayStore: https://play.google.com/store/apps/details?id=com.nianticlabs.pikmin
    • 홈페이지: https://pikminbloom.onelink.me/pWSt/refer

    보상 정보

    친구 초대로 가입하게 될 때 서로 받게 되는 보상 정보는 다음과 같습니다.

    초대받은 유저가 받는 보상

    • 꽃잎 20장과 정수 10개

    초대한 유저가 받는 보상

    • 처음 다섯 명까지 초대할 때마다: 추가 슬롯 1개
    • 이후 여섯 명부터 초대할 때마다: 정수 20개

    초대 코드 공유하는 방법

    • [프로필]을 탭합니다.
    • 좌측 상단의 [친구]를 탭합니다.
    • 우측 상단의 [초대]를 탭합니다.
    • [코드 공유]를 탭한 후, 원하는 방법으로 초대 코드를 공유합니다.
    • 초대 코드를 복사하고 공유를 합니다.
    • 초대 코드를 변경하고 싶은 경우에는 새로고침 버튼을 탭하면 됩니다.
    • 초대받은 유저가 초대 코드를 사용하여 게임 튜토리얼을 완료한 후에 보상이 지급됩니다. 또한 초대받은 유저는 자동으로 친구로 등록됩니다.

    자주 물어보는 질문 (FAQ)

    피크민 블룸 도움말에 있는 친구초대 관련한 FAQ를 대신 소개합니다.

    하나의 추천 코드를 사용해서 여러 명의 유저를 초대할 수도 있나요?

    • 추천 코드의 사용 유저 수에는 제한이 없습니다.

    내 초대 코드를 변경한 다음에는 예전 코드가 효력을 잃게 되나요?

    • 예전에 사용한 초대 코드는 효력을 읽습니다.

    내 초대 코드를 사용해서 어떤 유저가 앱을 시작했는데, 나는 보상을 받지 못했어요. 왜 그런가요?

    • 다음과 같은 이유를 생각해볼 수 있습니다. 초대받은 유저는 초대 코드를 재입력할 수 없다는 사실을 기억하세요.
    • 초대받은 유저가 초대 코드 입력을 깜박했어요.
    • 초대받은 유저가 다른 유저의 초대 코드를 사용했어요.
    • 초대받은 유저가 튜토리얼을 완료해서 자신의 보상을 수령하지 않았어요.
    • 초대하는 유저나 초대받는 유저의 친구 목록이 가득 찼어요(아래의 안내 참고).

    초대하는 유저나 초대받는 유저의 친구 목록에 빈자리가 없으면 어떻게 되나요?

    • 초대하는 유저나 초대받는 유저의 친구 목록에 빈자리가 없을 경우(Niantic의 모든 앱에 걸쳐, 최대 수인 400명에 도달한 경우), 다음 설명을 참고하세요.
    • 초대하는 유저와 초대받는 유저는 자동적으로 친구로 등록되지 않습니다.
    • 초대하는 유저는 보상을 받지 못합니다. 초대받는 유저는 보상을 받습니다.

    Ingress, Pokémon GO 등의 기타 Niantic 앱에서 이미 친구 사이인데요. 그래도 보상을 받을 수 있나요?

    • 초대받는 유저가 Pikmin Bloom을 처음으로 플레이한다면, 둘 다 보상을 받을 수 있습니다.

    보상 아이템의 최대 소지량을 초과해서도 보상을 받을 수 있나요?

    • 아니요.
