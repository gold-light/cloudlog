    히트2(Hit 2) 새벽부터 서버 포화, 서버 접속 대기 현상은 왜? 작업장 때문?

    2022-09-18 game / mobile / hit2

    서버 포화

    인기 많은 게임은 보통 서버가 포화상태가 되고, 접속 대기열이 생기는 것이 일반적이기는 합니다.
    하지만, 대부분 유저 접속이 많은 시간대에 한정되고, 새벽이나 아침에는 서버 접속이 원할한게 일반적인데 넥슨의 히트2는 그렇지 않아서 상당히 의문이 듭니다.

    평일은 학생들 학원이 끝나고, 직장인들 퇴근 시간 이후에 접속 대기가 생기고, 주말에는 점심즈음부터 접속이 어려워지는데, 히트2는 일요일 새벽부터 접속 대기가 생기고 있습니다.

    아래와 같이 일요일 오전 8시인데도 대부분의 서버가 포화 상태로 나타납니다.

    접속 대기

    이처럼 아침 일찍부터 접속하지 못하는 상황이 발생하고 있습니다.

    대체 이런 이유가 무엇일까 생각해봤는데, 아무래도 작업장 캐릭터 때문이 아닐까하는 추측이 듭니다.

    히트2가 자동사냥과 방치모드가 있기 때문에 다른 게임들보다 더 심한게 아닐까 싶은데, 실제로 게임에 접속해보면 여러 캐릭터가 한곳에 모여서 자동사냥하는 모습을 쉽게 볼 수 있습니다.

    그렇다면 넥슨에서는 작업장을 못 잡는 걸까요? 아니면 안 잡는 걸까요? 만약 작업장 캐릭터가 결제까지 하면서 게임을 하고 있다면 잡기 쉽지는 않을 것 같기도 합니다.

    뭐 어쨌거나 서버에 접속은 하게 해주셔야죠.
