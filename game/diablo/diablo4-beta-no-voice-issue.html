    디아블로4 (Diablo 4) 오픈 베타 캐릭터 음성이 나오지 않는 문제 해결 방법

    2023-03-24 game / pc / diablo

    개요

    디아블로4 (Diablo 4) 오픈 베타가 3월 25일 01시부터 시작되었는데, 게임 내에서 캐릭터 음성이 나오지 않는 문제가 발생했습니다.
    디아블로는 스토리 진행이 매우 중요하고 NPC 등의 사운드, 음성이 나오지 않으면 게임이 지루하고 재미가 없다고 느껴지기 때문에 반드시 해결해야 하는데, 해결 방법이 생각보다 간단합니다.

    배틀넷 런처 설정

    일단 게임을 종료하고 아래 스샷처럼 배틀넷 런처에서 [플레이] 버튼 옆에 있는 설정 아이콘( )을 클릭해서 [게임 설정] 메뉴를 선택합니다.

    음성 언어 변경

    [게임 설정] 화면에서 [텍스트 및 음성 언어]를 [English]로 변경합니다.


    • [English]를 선택하면 [언어를 변경하려면 새로운 데이터를 다운로드해야할 수도 있습니다]라는 메시지가 뜨는데, [확인] 버튼을 클릭하면 데이터가 다운로드 됩니다.


    • 데이터 다운로드가 완료되었으면 다시 [한국어]로 변경합니다.


    • 모두 끝났으면 오른쪽 아래쪽에 있는 [완료] 버튼을 클릭하고, 게임을 실행하면 됩니다.
