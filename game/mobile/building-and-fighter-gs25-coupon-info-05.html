    [빌딩앤파이터] GS25 도시락 쿠폰 나눔 5번째

    2023-12-11 game / mobile

    ToC on/off

    개요

    넥슨(Nexon)에서 서비스하고 있는 포켓몬고와 비슷한 위치기반 모바일 게임 [빌딩앤파이터]와 [GS25]가 함께 진행하는 100% 당첨 쿠폰 이벤트가 진행되고 있습니다. 지난번 방치형 모바일 게임 불레이드 키우기에 이어 도시락에 있는 쿠폰을 나눔하려고 이렇게 블로그에 공개하게 되었습니다. 쿠폰 번호는 아래쪽에 있습니다.

    GS도시락 프로모션 정보​

    도시락 프로모션 기간

    • 2023년 10월 26일(목) ~ 11월22일(수)​

    쿠폰 사용 기간 안내

    • 2023년 10월 26일(목) ~ 11월22일(수)​ 23:59 까지 ​

    제휴 도시락 상품명

    • 뭘좋아할지몰라다넣었어
    • NEW고진많도시락
    • 제주돼지두루치기정식
    • 11가지찬많은도시락
    • NEW정성가득비빔밥

    쿠폰 선물 아이템

    • 부하 영입카드 4개
    • 행동력 100개
    • 3단계 파워드링크 5개
    • 3,000 달러

    주의 사항

    1. 해당 이벤트 쿠폰은 각각 다른 난수번호로 계정당 3회까지 사용 가능합니다.
    2. 쿠폰 사용 시 아이템은 우편함으로 지급되며, 우편 도착까지 다소 시간이 소요될 수 있습니다.
    3. 쿠폰 사용 후 아이템이 지급되지 않는 경우 게임을 완전히 종료 후 재접속 부탁드립니다.
    4. 우편함으로 지급된 아이템의 보관 기관을 확인하시어 수령해 주시기 바랍니다.
    5. 회원코드를 잘못 입력하셨을 경우 복구가 불가하니, 회원코드 입력 전 정확한 정보를 꼭 확인해주시기 바랍니다.
    6. 이미 사용된 쿠폰이나, 유효기간이 지난 쿠폰은 사용할 수 없습니다.
    7. 이미 사용한 쿠폰은 아이템 회수나 복구 등의 도움을 드리기 어렵습니다.
    8. 쿠폰을 사용한 후 계정 초기화가 진행하여도 이미 사용한 쿠폰은 다시 사용하실 수 없습니다.
